Это часть RED HAWK

[ ОПИСАНИЕ ]

Этот каталог содержит в основном 4 файла, а именно

- admin.ini
- backup.ini
- others.ini
- readme.txt

Первые три файла являются обязательными для сканера RED HAWK. Эти файлы имеют список файлов и каталогов, которые сканер использует для сканирования.

[U S A G E & W A R N I N G S]

• Вы можете редактировать эти файлы, чтобы добавить свой собственный список.

• 3 разных INI-файла имеют список различных типов файлов и каталогов.
 - admin.ini: содержит список страниц администратора или каталогов администратора.
 - backup.ini: содержит список широко известных файлов резервных копий.
 - others.ini: в основном все остальные списки.

• ОБРАТИТЕ ВНИМАНИЕ, списки разделены COMMA "," любой другой используемый тип разделения вызовет ошибку при сканировании.
